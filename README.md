# Power-Bi-Sales-Dashboard

## 📊 Overview
This project is an interactive Power BI dashboard analyzing Madhav Store's e-commerce sales data to uncover business insights and trends.

## 🛠 Tools Used
- Power BI
- Excel / CSV
- DAX

## 📈 Key Insights
- Identified top-performing products and categories
- Analyzed monthly sales trends
- Found high-revenue regions
- Customer purchase behavior patterns
